# Email Phishing Detector
A machine learning project that analyzes email content to classify it as either legitimate or a phishing attempt.

Key Features:
Extracts and processes text features from raw emails using Natural Language Processing (NLP).
Utilizes a trained machine learning classification model to identify malicious patterns.
Evaluates model performance using standard metrics like accuracy and a confusion matrix.

Tech Stack:
Python
Scikit-Learn
NLTK / Pandas
